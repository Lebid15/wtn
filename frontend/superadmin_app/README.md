# Tenants App

تطبيق المستأجرين - تطبيق Next.js مع دعم متعدد اللغات

## البدء

تثبيت المكتبات:

```bash
npm install
```

تشغيل المشروع:

```bash
npm run dev
```

افتح [http://localhost:3001](http://localhost:3001) في المتصفح.

## التقنيات المستخدمة

- Next.js 14
- TypeScript
- Tailwind CSS
- next-intl (دعم متعدد اللغات)
- Radix UI
- Lucide Icons
